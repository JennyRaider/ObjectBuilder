public ObjectDefinitionBuilder objectVariable($scope, ObjectDefinition $objectDefinition)

Adds a variable into the class to represent the specified object.

For scope, specify "private", "protected" or "public". You may also prefix scope with "final" if desired.

This function will take the name of the specified object, strip out any generally undesired words (Interface, Class, etc.) and create a pretty variable name that will be defined in the class.

E.g.: An object named "UserRepositoryInterface" would be defined in the class as "$userRepository"
